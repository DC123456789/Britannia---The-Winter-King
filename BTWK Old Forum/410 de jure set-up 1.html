  1. Ols - 06-10-2014, 17:18
    So I think it's very clear that the Briton set up needs to be revised to fit 410 better. There are three key Briton kingdoms that we definitely need ingame in some respect, these are:

    - Icenia, which is most of Anglia for now. We need to reduce its area so that Cynwidion takes most of de jure Anglia
    - Cynwidion, which has a duchy but needs a kingdom, and might need local provinces reshaped to look better
    - Ceint, (or Cantauria) needs to be in Cantia and have two duchies of Rhegin and Ceint proper.

    There are also a few changes in 479 Briton areas:

    - Cornovia needs to be de jure
    - The Wetlands needs to be smaller, with Caer Lerion (in the Peak) reduced in size so that Lerion is the capital of a different duchy outside both.
    - The area around Alt Clut/Goddodin needs more provinces so that the borders look right in both 410 and 479.
    - We should probably add the Isles of Scilly, or Lyonesse, which was a larger single island in the period.

  2. DC123456789 - 06-10-2014, 18:34
    Yeah, that's basically what I was thinking. The Briton set-up for Anglia and Cantia should basically be the de jure setup in 410, since they're just restoring the destroyed kingdoms. Once we redraw the provinces there we could also have Caer Lundein as a full duchy.

    Dealing with de jure changes, I think that the simplest way would just be to make all the pre-existing Anglo-Saxon jarldoms (i.e. Grentbryce, Northfolc, Southfolc, East Seaxe, Cantware, The Weald) "formable" ones like the other ones, using the same mechanics. Their event flags would be set throughout the history so that they appear there at the appropriate times. This would require some fiddling around with the Angles, Saxons, and Jutes "titles" in the history, since that would trigger the events and make the de jure changes too quickly; perhaps they could be titular titles, like how the North Angles are currently set up?
