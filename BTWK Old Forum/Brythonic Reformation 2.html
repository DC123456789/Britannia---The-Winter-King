    Yes, let's put in a stripped down placeholder. I don't think we should tie ourselves down to a long project right now as I think it'd be nice to release another patch before the next DLC/official patch. Although if we add a reformed religion and then remove it later it could end up screwing things over.

    Let's concentrate on making the following reformable, in this order of priority:
    1) Brythonic
    2) Germanic
    3) Gaelic
    4) Pictsh

    Norse should probably be removed before the map extension and get merged into Germanic. Once the map extension is done I'd like to have included Basque Paganism and Ossetic Paganism (Alans). Tengriism would presumably also go in for the Huns. I've got quite a lot on my plate at the moment because I have a dissertation underway and because of the UK General Election, but I'll put in as much work as I can even if it's mainly advisory stuff/planning.

  3. DC123456789 - 19-03-2015, 18:23
    Norse only exists in the code currently, it's completely unused. I think that Germanic paganism could get split up after the map extension though, something similar to the lines of how we've split Celtic paganism (Norse, Frankish, and Anglo-Saxon-Jute-Frisian would make a pretty good split IMO).

    I don't think Ossentian Paganism is necessary; IIRC the Alans were Christianized fairly quickly after their migration into Gaul. Basque Paganism and Tengriism should obviously make it though.

    Also, for heads of religions, the Pontifex Maximus and Germanic head (Fylkir-esque title) are obviously going to be secular, inherited titles, but what about the Celtic religion heads? Ancient Religions gives them the Papal College of Cardinals succession, which is certainly interesting, but I'm not sure if this would work properly if applied for feudal rulers (since that's how we have druids currently).
