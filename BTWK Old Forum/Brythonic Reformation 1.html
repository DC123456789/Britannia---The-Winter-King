  1. DC123456789 - 03-11-2014, 18:53
    Okay, so here's how I'm thinking a custom reformation for the Brythonic religion could work:

    Reforming a religion sets off event 670, which is the actual event that converts all the provinces and characters (except for the reformer, I think). We could redirect the on_action to another event that only sets off event 670 for Brythonic pagans if the Old Gods have been restored (I'm assuming there's a global flag for this; if not, it would be fairly simple to put in). If they have been restored, then everything goes as usual; if not, then it would immediately undo all the stuff that reformation does (i.e. converting the reformer back to the original religion, making the original religion back into its own parent, deactivating the head of religion title) as well as re-enabling reformation through a special decision available to all Brythonic pagans (assuming we can't use some sort of hack to make the game continue to display the reform religion button). The decision would only be able to be taken once all the custom requirements and normal requirements have been fulfilled. This way we could have special requirements for a Brythonic reformation but all the other pagan religions could reform normally.

    Note that this is entirely theoretical and I haven't actually tried to see if this works yet.
