  1. JasperClay - 04-01-2016, 23:29
    Hi all - I just figured I'd make this discussion so that it exists. I want to gradually add in some events that allow the Warlord Chronicles main narrative to happen in the background in the book starts. I'm hoping merely to add in some event like the following for next release. (This will be in addition to a "forbid marriage" trait, which I figure on giving to Gwen, Arthur, Nimue, Morgan and Ceinwyn to start)

    This is just some pseudo-code that y'all can feel free to add to/critique/actually code. I'm posting it here to make sure there are no objections before I start messing with story events.
    Code:
    #Event 1. Standing in for the ending event in a Gundelus chain. Currently, just a hidden-window Gwen gets if Mordred becomes king, since Arthur shouldn't marry her before then.
character_event = {
	trigger = {
		character = guinevere
		k_dumnonia = { has_holder = mordred }
	}

	option = {
		character = arthur = { set_chracter_flag = winner_first_war }    #So, this event will eventually be triggered by Arthur winning a war vs. Siluria & Powys.
	}
}
#Event 2 - Should have nice localization, giving Arthur a choice between the virtuous Ceinwyn and the entrancing Gwen. 
character_event = {
	
	trigger = {
		character = arthur
		year = 480
		k_dumnonia = { 
			holder = { 
				mordred = {
					regent = { ROOT }
				}
			}
		}
		had_character_flag = { winner_first_war = { days = 365 } }
	}

	immediate = {
		character = { guinevere = { remove_trait = forbid_marriage } }
		character = { ceinwyn = { remove_trait = forbid_marriage } }
		character = { arthur = { remove_trait = forbid_marriage } }
	}

	optionA = {
		ai_weight = 90
		trigger_event = { id = 312102 }    #Arthur/Gwen marriage
		character = { Ceinwyn = { 
			add_trait = forbid_marriage    #Refuses to marry anyone 
			add_lover = derfel                 #But basically marries Derfel
			}
                #trigger_event = { Start Round 2 with Powys. }
		}
	}

	optionB = {
		ai_weight = 10
		add_spouse = ceinwyn
		character = { derfel = {
			add_lover = nimue    #This is how we deal with lover musical chairs. Also, makes it likely Derfel has potential decent Brythonic heirs if he becomes King of Icenia.
			}
		}
                #Possible event sending Guinevere fleeing to Benoic, and adding Lancelot as her lover (event for Lancelot, too)? or something else? Just leave her?
	}
}
    That's just a starter to get some of the proper marriages happening from the 479 start. Ideally, first a 479 -> Gundelus chain would be written, connecting to the Uther events on one end, and then this series on the other. Then, moving from here, Lancelot gets added to the mix, tying into Derfel's Wolftails events, and the fall of Benoic -> 486.

  9. JasperClay - 07-01-2016, 01:46
    Okay, just posting this recoded version so you guys can take a quick look. No localization written yet. As the story events expand, I plan to change this one, but for next release, this event, properly localized for Arthur and the accompanying forbid_marriage traits are all I have as minimum goals.

    Code:
    character_event = {	# Event for Arthur - Guinevere Marriage
	id = 312102
	desc = "EVTDESC312102"
	picture = GFX_evt_council
	
	only_rulers = yes
	is_triggered_only = yes

	trigger = {
		year = 480			#Eventually gate to make sure Gundelus events happen first
		character = 3 		#Event fires for Arthur
		is_married = no
		top_liege = {
			has_landed_title = k_dumnonia
			regent = {
				character = ROOT
			}
			character = 5 		#Mordred is independent, Arthur's liege, King, and Arthur is regent. 
		}
	}
	
	immediate = {
		fertility = 0.55
		remove_trait = forbid_marriage	#Arthur
		any_playable_ruler = {
			any_courtier = {
				limit = {
					character = 26 		#Guinevere
				}
				remove_trait = forbid_marriage
				fertility = 0.35
			}
		}
		any_playable_ruler = {
			any_courtier = {
				limit = {
					character = 10 		#Derfel
				}
				remove_trait = forbid_marriage
			}
		}
		any_playable_ruler = {
			any_courtier = {
				limit = {
					character = 27 		#Ceinwyn
				}
				remove_trait = forbid_marriage
			}
		}
		any_playable_ruler = {
			any_courtier = {
				limit = {
					character = 31 		#Lancelot
				}
				remove_trait = forbid_marriage
			}
		}
	}
	
	option = {
		name = "EVTOPTA312102"
		if = {
			limit = {
				any_playable_ruler = {
					any_courtier = {
						character = 26		#Guinevere
						is_married = no
					}
				}
			}
			ROOT = { 				#Arthur
				add_spouse = 26		#Guinevere
				add_lover = 26		#Guinevere
			}
		}
		if = {
			limit = {
				any_playable_ruler = {
					any_courtier = {
						character = 27	#Ceinwyn
						is_married = no
					}
				}
			}
			any_playable_ruler = {
				any_courtier = {
					limit = { character = 27 }	#Ceinwyn
					add_spouse = 10 	#Derfel
					add_lover = 10 		#Derfel
				}
			}
		}
	}
	
	option = {
		name = "EVTOPTB312102"
		if = {
			limit = {
				any_playable_ruler = {
					any_courtier = {
						character = 27	#Ceinwyn
						is_married = no
					}
				}
			}
			ROOT = { 				#Arthur
				add_spouse = 27 	#Ceinwyn
			}
		}
		if = {
			limit = {
				any_playable_ruler = {
					any_courtier = {
						character = 26	#Guinevere
						is_married = no
					}
				}
			}
			any_playable_ruler = {
				any_courtier = {
					limit = { character = 26 }	#Guinevere
				} 
				add_spouse = 31 	#Lancelot
				add_lover = 31 		#Lancelot
			}
		}
		if = {
			limit = {
				any_playable_ruler = {
					any_courtier = {
						character = 6	#Nimue
						is_married = no
					}
				}
			}
			any_playable_ruler = {
				any_courtier = {
					limit = { character = 6 }	#Nimue
				} 
				remove_lover = 2 	#Merlin
				add_lover = 10 		#Derfel
			}
		}
	}
}
    I thought there might be much more elegant ways to do this, but it's late, and I'm more interested in getting a working version than writing localization tonight so I can test various levels of elegance in scopes and if statements ad nauseum. Please feel free to repost greater elegance, if you like, and I'll learn from the corrections.
