  9. DC123456789 - 08-07-2015, 21:38
    All of the other known Frankish kings were murdered and taken personally be Clovis, actually - it could be like the Charlemagne/Carloman vanilla event I suppose. The thing is, this didn't happen until around 507, which would have been 20 years after they first became subservient to him, and that's the main thing I want to replicate ingame right now.

    Salians and Ripuarians have to be seperate kingdoms in order to represent the division between Soissons and Metz post-Clovis, which were essentially based on those two precursor kingdoms and are currently represented by the same title. Clovis (or rather Childeric) starting off as duke-level with a way to work his way up to king-level could work, I guess.

    The way namespaces work is that each namespace is allocated one million event ids, with the first one (alphabetically) being given 1000000-1999999, the second 2000000-2999999, etc. This works in vanilla where most event ids stay in the ten thousands, but the problem is that we have quite a few events with event ids over 999999. Most were written before namespaces were introduced; they usually come from adding a 0 to the end of an event id in order to link them in a way that makes sense (e.g. a titular title event and its corresponding removal event) or adding in an event into a pre-existing sequence (so for example adding in an event between 313506 and 313507, 3135060, 3135061, 3135062, etc. would be used). If, say, we had an event 1000001 and an event AAA.1, the AAA.1 would become 1000001 in the code, causing conflicting event ids. We were fine up to them because the first conflict won't come up until the number of namespaces goes up to somewhere in the 30s. One possible solution would be to find all the culprits in renumber them, but there are quite a bit (check the Validator) and in my opinion not worth the effort. To be honest I don't really see the point of a lot of internal namespaces anyway; there are plenty of "normal" event ids and domains that are still available to use. Namespaces should be reserved for integrating mods and DLC events IMO.
